# Scrape
All the News That's Fit to Scrape.

This web app scraps the latest news from "reforma.com" and lets users view and leave comments on them. 

This project uses:

- HTML
- CSS
- Bootstrap
- Javascript
- JQuery
- Ajax
- Node JS
- Express
- Handlebars
- Mongo DB
- Mongoose
- Cheerio
- Axios
- MVC methodology (Model View Controller)
